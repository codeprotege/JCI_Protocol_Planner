# JCI Seating Design Context

## Design Context

### Users
JCI members and event protocol teams preparing formal seating for ceremonies, official visits, and recognition-heavy functions. They need to move from a raw attendee roster to a defensible seat order quickly and with confidence.

### Brand Personality
Professional, brutalist, clean.
The interface should feel authoritative, fast, and operational rather than decorative.

### Aesthetic Direction
Support both light and dark themes. Use a sharp, structured, protocol-desk visual language with strong contrast, rigid layout lines, disciplined typography, and zero soft pastel treatment. Avoid pink, pastel palettes, and playful consumer-app styling.

### Design Principles
1. The seat order must read like an official decision, not a casual suggestion.
2. Inputs should feel procedural and fast for operators under event pressure.
3. Priority logic should be visible enough to explain outcomes without cluttering the main flow.
4. Light and dark themes should both feel formal and controlled.
5. The interface should look strict and clear, not ornamental.
